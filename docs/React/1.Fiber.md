## 什么是 Fiber

- `Fiber`其实指的是一种数据结构，它可以用一个纯 JS 对线来表示(虚拟 dom)：

```js
const fiber = {
  type,
  props,
  ref,
  children,
};
```

- `Fiber`是一个执行单元，每次执行完一个执行单元，React 就会检查现在还剩多少时间，如果没有时间就将控制权让出去。

## Fiber 关键特征

- 增量渲染
- 暂停，终止，复用渲染任务
- 不同更新的优先级
- 并发方面新的基础能力
